## 상속 구조에서 Builder 생성 오류

### 문제
엔티티 생성 시 생성자 파라미터가 많아져 Builder 패턴을 적용하는 과정에서, `BaseEntity`를 상속하는 엔티티에 `@Builder`를  
적용했을 때 빌더 생성 오류가 발생했습니다.  
상속 구조를 가진 엔티티에서 객체 생성 코드가 정상적으로 동작하지 않았습니다.

### 해결
원인을 확인한 결과, Lombok의 `@Builder`는 상속 구조를 고려하지 않기 때문에 부모 필드까지 포함한 빌더 생성에 한계가 있었습니다.  
이를 `@SuperBuilder`로 변경해 상속 구조에서도 빌더가 정상적으로 생성되도록 수정했습니다.

### 결과
상속 구조를 가진 엔티티에서도 빌더 패턴을 안정적으로 적용할 수 있게 되었습니다.  
이 경험을 통해 Builder 패턴 적용 시에는 엔티티 구조, 특히 상속 여부를 함께 고려해야 한다는 점을 알게 되었습니다.
