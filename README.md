# 📌 Novelia

Spring Boot, Rest API, JPA 기반으로 구현한 **웹소설 플랫폼 개인 프로젝트**입니다.<br>
**Spring Boot, Rest API, Spring Security(JWT 기반 인증), JPA, MySQL**을 활용하여 회원 관리, 작가 권한 요청 관리, 소설 관리, 회차 관리, 회차 구매 기능을 구현했습니다.

## 💻 개발 환경

- ☕ **Java 17**
- 🍃 **Spring Boot 3.5.11**
- 📡 **RestAPI**
- 🔐 **Spring Security**
- 🪪 **JWT**
- 🗃️ **JPA**
- 🐬 **MySQL**
- ⚙️ **Gradle**
- 🔧 **Git / GitHub**
- 💡 **IntelliJ IDEA**

## ✨ 주요 기능

### 👤 회원 관리

#### 회원가입
- 사용자는 아이디와 비밀번호를 입력하여 신규 회원으로 가입할 수 있습니다.
- 아이디는 **6~12자 사이의 영소문자 조합**이어야 합니다.
- 아이디는 **중복될 수 없습니다.**
- 비밀번호는 **8~20자 사이의 특수문자 조합**이어야 합니다.

#### 로그인
- 사용자는 회원가입 시 등록한 아이디와 비밀번호로 로그인할 수 있습니다.
- 등록한 아이디 또는 비밀번호가 일치하지 않으면 로그인에 실패합니다.
- 로그인에 성공하면 **JWT 기반의 액세스 토큰**이 발급됩니다.

---

### ✍️ 작가 권한 요청 관리

#### 작가 권한 요청
- 일반 사용자는 관리자에게 **작가 권한 요청**을 할 수 있습니다.

#### 작가 권한 승인 / 거절
- 관리자는 일반 사용자의 작가 권한 요청을 **승인 또는 거절**할 수 있습니다.

---

### 📚 소설 관리

#### 소설 목록 조회
- 공통적으로 **카테고리 필터링**, **제목 또는 소개 검색**, **페이징 기능**을 제공합니다.
- 비회원을 포함한 모든 사용자는 **삭제되지 않은 소설 목록**을 조회할 수 있습니다.
- 작가는 **본인이 등록한 소설 목록**을 조회할 수 있습니다.
- 관리자는 **삭제된 소설을 포함한 모든 소설 목록**을 조회할 수 있습니다.

#### 소설 등록
- 소설 등록은 **작가만 가능**합니다.
- 작가는 **섬네일 이미지, 제목, 소개**를 입력하여 새로운 소설을 등록할 수 있습니다.

#### 소설 상세 조회
- 비회원을 포함한 모든 사용자는 **삭제되지 않은 소설**의 상세 정보를 조회할 수 있습니다.
- 작가는 **본인이 등록한 소설**의 상세 정보를 조회할 수 있습니다.
- 관리자는 **삭제된 소설을 포함한 모든 소설**의 상세 정보를 조회할 수 있습니다.

#### 소설 수정
- 소설 수정은 **작가만 가능**합니다.
- 작가는 **본인이 등록한 소설에 한해서** 섬네일, 제목, 소개를 수정할 수 있습니다.

#### 소설 삭제
- 작가는 **본인이 등록한 소설에 한해서** 삭제할 수 있습니다.
- 관리자는 **모든 소설을 일괄 삭제**할 수 있습니다.
- 소설 삭제는 **논리 삭제(Soft Delete)** 방식으로 처리합니다.

---

### 📖 회차 관리

#### 회차 목록 조회
- 공통적으로 **페이징 기능**을 제공합니다.
- 비회원을 포함한 모든 사용자는 **삭제되지 않은 회차 목록**을 조회할 수 있습니다.
- 작가는 **본인이 등록한 소설의 회차 목록**을 조회할 수 있습니다.
- 관리자는 **삭제된 회차를 포함한 모든 회차 목록**을 조회할 수 있습니다.

#### 회차 등록
- 회차 등록은 **작가만 가능**합니다.
- 작가는 **본인이 등록한 소설에 한해서** 회차 제목과 내용을 입력하여 새로운 회차를 등록할 수 있습니다.

#### 회차 상세 조회
- **로그인한 일반 사용자**만 회차 상세 조회가 가능합니다.
- 사용자는 **삭제되지 않은 회차**의 상세 정보를 조회할 수 있습니다.
- 단, **해당 회차를 구매하지 않은 경우 열람할 수 없습니다.**

#### 회차 수정
- 회차 수정은 **작가만 가능**합니다.
- 작가는 **본인이 등록한 회차에 한해서** 제목과 내용을 수정할 수 있습니다.

#### 회차 삭제
- 회차 삭제는 **작가만 가능**합니다.
- 작가는 **본인이 등록한 회차에 한해서** 회차를 삭제할 수 있습니다.

---

### 💳 회차 구매 관리

#### 회차 구매
- 회차 구매는 **로그인한 일반 사용자**만 가능합니다.
- 사용자는 **삭제되지 않은 회차**를 구매할 수 있습니다.

---

## 🔐 권한 정책

- **USER**: 로그인한 일반 사용자
- **AUTHOR**: 작가 권한 사용자
- **ADMIN**: 관리자 권한 사용자
